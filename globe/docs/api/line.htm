Line Properties

Example (Line Properties)

locations (array of lat/lng objects)

[]

You must provide at least to two locations (start and end point).

offset (float)

0.01

Distance to the earth surface.

offsetFlow (float)

0

Distance to the earth surface at the end point.

offsetEasing (easing)

arc

An easing function that describes the offset from start to end.

color (color)

#FF0000

hairline (boolean)

false

width (float)

1

Only available if hairline = false.

endWidth (float)

-1 (same value as width)

Width at the end point.
Only available if hairline = false.

clip (float)

1

0.0 - 1.0

Displays only a part of the line. A clip value of 0.5 displays only the first half of the line.

dashed (boolean)

true

dashSize (float)

0.5

dashRatio (float)

0.5

0.01 - 0.99

dashOffset (float)

0

A dashed line is rendered as a transparent object.

visible (boolean)

true

Advanced:

opacity (float)

1

0.0 - 1.0

transparent (boolean)

Needs to be true for opacity, dashed lines or if you want to overlay transparent markers/lines with alwaysOnTop.

alwaysBehind (boolean)

false

The line is placed behind other lines/markers of the same group (opaque or transparent).

alwaysOnTop (boolean)

false

The line is placed on top of other lines/markers of the same group (opaque or transparent).

earth (EarthInstance)

Reference to the EarthInstance.

object3d (THREE.Object3D)

About THREE.js

 
You can add custom properties that are copied over to the LineInstance.
