API: Class Animation

Initiate

Initiate with the animate() method of Earth, Marker, Line or Overlay.

Animation Properties

Example (Animatable Properties)

duration (float)

400 (ms)

relativeDuration (float)

0

Adds a duration relative to difference from start to end value. For lat/lng properties the relativeDuration is added as milliseconds per 1000 kilometers distance from start to end location.

easing (easing)

in-out-quad

See Easing

loop (boolean)

false

If true the animation never completes and jumps back to the start value.

oscillate (boolean)

false

If true and loop=true the animation plays back and forth.

lerpLatLng (boolean)

false

lerpLatLng only effects lat/lng properties.

If lerpLatLng is true the lat/lng values are lerped as numbers (avoiding the polar regions) instead of the 3d position (shortest path).

complete (function)

Function to call when the animation completes.

end (function)

Function to call when the animation completes or it is stopped.

target (Earth, Marker, Line or Overlay)

Reference to the animated instance.

t (float)

0.0

0.0 - 1.0

Current time of the animation as value from 0 to 1.

 
There are special properties (approachAngle and zoom) that are only used by the earths goTo method.

Animation Methods

stop( dispatchComplete, jumpToEnd )

dispatchComplete
(boolean) default: false | Call the complete function?
jumpToEnd
(boolean) default: false | Set the property to the end value immediatly.

Easing

Example (Animation Easing)

linear

No easing

in-quad, in-cubic, in-quart

The animation starts slowly.

out-quad, out-cubic, out-quart

The animation ends slowly.

in-out-quad, in-out-cubic, in-out-quart

The animation starts and ends slowly.

in-back, out-back, in-out-back

The animation overshoots start and/or target value.

elastic

Elastic effect.

bounce

Bounce effect.

arc

Special easing function that reaches the target value at half time and returns to the start value. This easing function is useful to draw lines as arcs with offsetFlow and offsetEasing.

 
You can register your own easing functions. 
Earth.Animation.Easing['my-special-easing'] = function ( t ) {
	// t (time) is a value from 0 to 1 you need to modify here
	return t;
};
